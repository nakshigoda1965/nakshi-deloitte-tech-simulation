# Deloitte Australia Technology Job Simulation

This project was completed as part of the Deloitte Australia Technology Virtual Experience Program on Forage.

## Overview
The project converts two different Industrial IoT (IIoT) telemetry data formats into a unified JSON structure.

## Features
- Converts Format 1 to the unified schema.
- Converts Format 2 to the unified schema.
- Converts ISO 8601 timestamps to Unix epoch milliseconds.
- Includes automated unit tests.

## Technologies
- Python
- JSON
- datetime
- unittest
